### 🚀 What I'm building

- **Jebrin** — a programmable gateway that decides, for every web request, whether to allow, block, or **charge AI crawlers** via USDC micropayments (HTTP 402 / x402). Giving site owners control over bots *and* a way to monetize them.
- **Diannao** (电脑, "electric brain") — a multi-agent *idea-to-shipped* pipeline. Capture a half-formed thought and it's classified by an LLM and routed through a filesystem workflow with a human approval gate. Dockerized Python/Flask with a live dashboard, deploying to AWS EC2.
